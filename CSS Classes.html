Pseudo‑Classes Demo — :hover, :focus, :active, :nth-child()

Examples applied to links, buttons and list items. Use Tab / mouse to try focus, hover, and active states.

:hover, :focus, :active on a link
Visit docs

:hover — changes background and lifts the link when the mouse is over it. :focus — a visible focus ring appears when tabbed to (keyboard navigation). :active — pressed state when you click.

:hover, :focus, :active on a button

Buttons show subtle lift on :hover, a focus ring on :focus (for keyboard users) and a pressed look on :active while the mouse is held down.
